## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Navasakthivel/The-Developers-Arena-Weekly-Task7.git
```

### Navigate to Project

```bash
cd The-Developers-Arena-Weekly-Task7
```

### Install Dependencies

```bash
npm install
```

### Run Project

```bash
npm start
```

or

```bash
npm run dev
```
